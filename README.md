# Deskripsi Proyek
Proyek game bernama JAN PEKERJA . Game ini memiliki 6 map + 1 map utama.
Didalam game ini, player diharuskan melakukan pekerjaan agar dapat survive. 
Player dapat melakukan pekerjaan di cave,forest,farm dan meathouse.
Setiap pekerjaan terdapat levelnya masing-masing dan memiliki reward yang berbeda. 
Player juga diharuskan untuk melakukan jual beli item di market agar dapat menjaga status bar yang dimiliki.
Banyak aktivitas lain yang dapat dilakukan oleh player selain bekerja , seperti tidur, mandi, bermain , dan yang lainnya , tentu dengan manfaatnya masing - masing.
Kemudian game akan berakhir jika player memiliki health 0 . 

## Map
- Main map
- Cave
- Farm
- Meathouse
- Forest
- Home
- Market

## Status Bar
- Health
- Energy
- Hunger
- Hygiene
- Happiness

## Rules
- Dilarang melakukan back ketika game sedang berjalan
- Dilarang melakukan cheat atau merubah game
- Dilarang melakukan skip level
- Diwajibkan mengisi nama di menu utama
- Diwajibkan melakukan pekerjaan agar dapat survive didalam game.
